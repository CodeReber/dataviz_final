Predicting Polar Bear Habitat Selection

Machine Learning Pipeline Overview

Resource Selection Function (RSF) is a measurement of how conducive the environment/habitat is for long-term survival. We generated a model to explore the impact of climate on polar bear selection of habitat

  1. Import and clean the dataset
  2. Inital steps (not shown): linear and logistic regression revealed very weak correlation
  3. Converted RSF into categorical variable for low, mid, and high RSF values
  4. Split data into training and testing data set (20% testing)
  5. Normalize quantitative data using MinMaxScalar
  6. Random Forest: Hypertuned random forest score 0.63; used model for prediction
Machine Learning Retrospective

The limited dataset proved difficult for identifying a direct correlation between climate and RSF values. Converted to categorical resulted in a fairly good model for predicting RSF based on climate. A neural network may provide a more accurate model. Initial analysis reveals that climate change (high temperature, high CO2 emissions, and low sea ice extent) result in low RSF values indicating either poolar bears inability to acclimate to temperature changes or decreases in optimal habitat.

Predicting Polar Bear Mobility

Machine Learning Pipeline Overview

Calculated total distance traveled per month for each bear to assess impact of climate on polar bear activity.

  1. Import and clean the dataset
  2. Average distance traveled calculatd from lat/long coordinates and Haverstine Algorithm
  3. Converted mobility (avg. distance traveled) into categorical distance traveled - high, mid, low
  4. Assign features and target values then split data into training and testing datasets (testing=30%)
  5. Normalize quantitative data using MinMaxScalar
  6. Multiple Linear Regression: No significant correlation between climate metrics and den location
  7. Logistic Regression: Scored 0.2, not a significant correlation.
  8. Random ForestSea Ice Extent was ranked highest in feature importance. Hypertuned model scored 0.56.
  9. kNN: Range of k values revealed n_neighbors = 5 to have highest accuracy with a score of 0.63. Used model for prediction
Machine Learning Retrospective

This model was by far the most successful at categorizing polar bear activity based on climate metrics. While standard linear/logistic regression did not reveal a strong correlation, transforming mobility into categorical values (low, mid, high) enabled us to develop a fairly accurate model.

K-Means Labeling of Arctic Zones

Unsupervised K-Means Clustering of Location Overview

Utilize K-Means clustering to of latitudinal and longitudinal data for polar bear location from 2009 - 2016.

  1. Import and clean the dataset -- remove null values, select relevant columns, remove outliers
  2. Plot coordinates onto a map for visualizing "zones" or k clusters
  3. Select k based on a high score with useful resolution for locating polar bears across a massive surface area.
  4. Create, train and fit the model based on k clusters using the 2009-2016 dataset
  5. Apply our model to the 1985-2016 polar bear dataset for final machine learning algorithms

Predicting Polar Bear Den Location

Machine Learning Pipeline Overview

By applying our k-means clustering we developed a model to use predict location of polar bear dens based on climate metrics.

  1. Import and clean the dataset
  2. Split data into training and testing data set (20% testing)
  3. Normalize quantitative data using MinMaxScalar
  4. Multiple Linear Regression: No correlation between climate metrics and den location
  5. kNN: Range of k values revealed n_neighbors = 5 higher accuracy (score = 0.2)
  6. Random Forest: Hypertuned random forest score 0.2; used model for prediction
Machine Learning Retrospective

The dataset hindered our ability to create a successful model when only assessing climate as a predictor of den & polar bear location. Previous models incorporated the month and year and resulted in a strong random forest model scoring 0.75. Based on app continuity, we chose to exclude month and year from our prediciton but future models should include this critical feature.
